Процедура ОчиститьНесуществующиеСсылкиВТЧ(Объект, ИмяТЧ, ИменаКолонок)
	Если ТипЗнч(ИменаКолонок) = Тип("Строка") Тогда
		Колонки = СтрРазделить(ИменаКолонок, ",", Ложь);
	ИначеЕсли ТипЗнч(ИменаКолонок) = Тип("Массив") Тогда
		Колонки = ИменаКолонок;
	Иначе
		Возврат;
	КонецЕсли;		
	
	Для Каждого СтрокаТЧ Из Объект[ИмяТЧ] Цикл
		
		Для Каждого ИмяКолонки Из Колонки Цикл
			Если ЗначениеЗаполнено(СтрокаТЧ[ИмяКолонки]) И НЕ ОбщегоНазначения.СсылкаСуществует(СтрокаТЧ[ИмяКолонки]) Тогда
				СтрокаТЧ[ИмяКолонки] = Неопределено;
			КонецЕсли 
		КонецЦикла;
		
	КонецЦикла;
КонецПроцедуры
